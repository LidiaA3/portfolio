Desarrollo web de una web musical

Desarrollo web de una página completa para un grupo de música ficticio. El proyecto cuenta con un total de seis páginas principales acompañadas de algunas subpáginas. Todo el proyecto ha sido desarrollado mediante html, css y javascript. Para el correcto funcionamiento de todos los elementos y para asegurar un poco de calidad en el resultado, todas las páginas se han basado en Bootstrap. El resultado final es una web completa y dinámica con un aspecto profesional.
